# Hospital Patient Data Analysis and Data Preprocessing

## 📌 Project Overview

This project focuses on analyzing and preprocessing hospital patient records and billing information using Python. The objective is to clean, transform, and integrate patient and billing datasets to generate meaningful insights related to department-wise revenue and doctor performance.

---

## 🎯 Objectives

* Load and explore hospital patient datasets.
* Select relevant columns required for billing analysis.
* Remove unnecessary administrative information.
* Handle missing values in billing records.
* Identify and remove duplicate patient records.
* Merge patient and billing datasets.
* Concatenate additional patient and billing information.
* Prepare a clean dataset for further analytical tasks.

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Jupyter Notebook

---

## 📂 Dataset Description

The dataset contains hospital patient information, including:

* Patient ID
* Department
* Doctor Details
* Diagnosis Information
* Bill Amount
* Administrative Details

An additional billing dataset is also used to enrich patient billing records.

---

## 🔄 Data Preprocessing Tasks

### 1. Dataset Exploration

* Loaded the patient dataset.
* Examined dataset structure using `info()`.

### 2. Column Selection

Selected billing-related columns:

* PatientID
* Department
* Doctor
* BillAmount

### 3. Column Removal

Removed unnecessary administrative columns such as:

* ReceptionistID
* CheckInTime

### 4. Department-wise Revenue Analysis

Used `groupby()` to calculate total bill amounts generated by each department.

### 5. Duplicate Handling

Removed duplicate patient records based on `PatientID` to ensure data consistency.

### 6. Missing Value Treatment

Filled missing `BillAmount` values using the mean bill amount.

### 7. Dataset Integration

Merged patient and billing datasets using `PatientID` as the common key.

### 8. Data Concatenation

* Added new patient records row-wise.
* Added new billing-related columns column-wise.
